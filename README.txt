Manage custom CSS and JavaScript snippets with header/footer/<body> placement control.

== Description ==
Easily add custom CSS and JavaScript to your WordPress site with options to:
- Insert code in header or footer or After <body>
- Toggle active/inactive status
- Set execution priority (premium)
- Page-specific targeting (premium)
- Enable Framework (Bootstrap, Bulma, Tailwind) (premium)

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/custom-scripts-manager`
2. Activate through WordPress admin
3. Navigate to Scripts Manager in admin menu
